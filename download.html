Installation Guide

Easy setup in just a few steps

Step 1: Download

Click the download button above to get the latest Xenia Canary build from the official download page. The file will be in a compressed ZIP format.

Step 2: Extract

Extract the downloaded ZIP file to a folder of your choice. We recommend creating a dedicated folder like "C:\Xenia" for easy management.

Step 3: Install Dependencies

Make sure you have the latest Visual C++ Redistributable installed. The emulator requires this to run properly. Download it from Microsoft's website if needed.

Step 4: Run Xenia

Launch xenia_canary.exe from the extracted folder. The emulator will create necessary configuration files on first launch.

Step 5: Configure Settings

Configure your controller, graphics API (Vulkan or DirectX 12), and audio settings through the emulator's configuration menu.

Step 6: Load Games

Use File > Open to load your Xbox 360 game files. Supported formats include ISO, XEX, and XBLA archives. Check game compatibility before playing.
